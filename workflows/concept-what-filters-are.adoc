= 필터란
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
워크플로우에서 WFA 필터를 사용하여 필요한 리소스를 선택할 수 있습니다.

WFA 필터는 WFA 데이터베이스에서 작동하는 SQL 기반 쿼리입니다. 각 필터는 특정 사전 형식의 요소 목록을 반환합니다. 반환된 요소는 SQL 쿼리에 지정된 선택 조건을 기반으로 합니다. 필터를 만들거나 편집하려면 SQL 구문을 알고 있어야 합니다.
