= 워크플로 미리보기 대화 상자
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
워크플로 미리 보기 대화 상자에서는 워크플로와 관련된 사용자 입력 값을 지정한 다음 지정된 값을 사용하여 해당 워크플로의 실행을 미리 볼 수 있습니다.



== 사용자 입력 값

워크플로 미리 보기 대화 상자에는 선택한 워크플로와 관련된 사용자 입력이 표시되고 사용자 입력의 값을 설정할 수 있습니다.



== 명령 버튼

* * 미리보기 *
+
연결된 워크플로의 모니터링 창을 엽니다.

* * 취소 *
+
대화 상자를 닫습니다.
