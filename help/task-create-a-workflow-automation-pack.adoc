= WFA 워크플로우 팩을 생성합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
스토리지 자동화 및 통합 요구사항에 맞는 WFA(OnCommand Workflow Automation)에서 워크플로우 팩을 생성할 수 있습니다.

.단계
. 웹 브라우저를 통해 * WFA * 창에 로그인합니다.
. 콘텐츠 관리 * > * 팩 * 을 클릭합니다.
. 새 팩 * 아이콘을 클릭합니다.
. 새 팩 * 대화 상자에서 * 이름 *, * 작성자 *, * 버전 * 및 * 설명 * 필드에 값을 입력합니다.
. 저장 * 을 클릭합니다.
. 새 팩이 * Packs * 창에 생성되었는지 확인합니다.
