= 새 팩 대화 상자
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
새 팩 대화 상자에서 새 팩을 만들 수 있습니다.

* * 이름 *
+
이름을 입력하고 팩을 저장할 수 있습니다.

* * 버전 *
+
버전을 입력하고 팩을 저장할 수 있습니다.

* * 작성자 *
+
작성자 이름을 입력하고 팩을 저장할 수 있습니다.

* * 설명 *
+
설명을 입력하고 팩을 저장할 수 있습니다.





== 명령 버튼

* * 저장 *
+
팩을 저장하고 대화 상자를 닫습니다.

* * 취소 *
+
저장하지 않고 대화 상자를 닫습니다.
